# UX Analysis: TV Food Maps
URL: https://www.tvfoodmaps.com

## UX Patterns Extracted
- **Search and Filter mechanism:** Centralized search focused on location ("City, State") alongside a very prominent "Near Me" button using geolocation. It offers dedicated entry points for "Find By Show" and a "Road Trip Builder".
- **Multi-show context:** Rather than just one show, the site groups content into horizontal scrolling lists or grids of shows (e.g., "Diners, Drive-ins and Dives", "The Bear: Featured Restaurants", "Best Thing I Ever Ate").
- **Restaurant card fields:** Clean card layout featuring a large image, restaurant name, city/state, a brief description of its TV appearance, a "TVF Score" (e.g., TVF 100.0), and a pill/tag indicating which show featured it.
- **"Near me" feature:** Positioned prominently next to the main search bar as an icon/button, encouraging location-based discovery as a primary use case.
- **Trip planning:** Offers a dedicated "Road Trip Builder" allowing users to string together multiple TV restaurant stops along a route.
- **AI Integration:** Features an "AI Concierge" promising personalized suggestions based on "vibe".
