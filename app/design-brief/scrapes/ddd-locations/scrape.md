# Scrape Results: DDD Locations
URL: https://dinersdriveinsdiveslocations.com

## Site Focus
Search/filter UX, state browsing, restaurant card layout, mobile nav.

## UX Patterns

### 1. Navigation & Structure
- Header contains links for: by CITY, by STATE, by FOODS, by CUISINE, ROAD TRIP, RECIPES, SHOP.
- Mobile nav collapses into a hamburger menu.
- State-specific pages list restaurants by city.
- Breadcrumbs or easy "Back to Home" links (logo).

### 2. Search & Filtering
- Home page has large buttons for "SEARCH DINERS by STATES" and "SEARCH DINERS by CITIES".
- State pages have a "DDD [State] CITY LIST" dropdown.
- "Trending [State] Cities" links (e.g., Los Angeles, San Diego, San Jose for California).
- "Sort by Cities" visual cues (arrows).

### 3. Restaurant Card Layout
- Restaurant Name (Link to detail).
- Image of the restaurant/food.
- Full address (Street, City, State, Zip).
- Phone number.
- City link (e.g., Baker, CA).

### 4. Visuals & Branding
- Color Palette: Red, White, Blue/Navy header.
- Typography: Bold sans-serif for headings.
- Heavy use of icons (map pins, search icons).
- Ad-heavy layout (interstitials, banners).

### 5. Mobile Responsiveness
- Grid layout collapses to single column.
- Text sizes remain readable.
- Hamburger menu for top-level nav.
- Video/Ad overlays can be intrusive.
